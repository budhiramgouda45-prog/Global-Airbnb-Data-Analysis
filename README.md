# Global-Airbnb-Data-Analysis


-I developed a Global Airbnb Performance Dashboard using Power BI to analyze Airbnb listing data across different locations. The main objective of this project was to understand pricing trends, customer preferences, and overall business performance.

I started by collecting and cleaning the dataset in Excel, where I handled missing values and ensured data consistency. Then I imported the data into Power BI and performed data modeling to establish relationships between tables.

I created interactive dashboards to visualize key metrics such as total listings, average price, and revenue. I also built visualizations to analyze location-wise performance, room type distribution, and pricing trends over time.

Through this project, I identified that certain cities generate higher revenue due to higher demand and pricing, and I also observed differences in customer preferences for room types.

This project helped me improve my skills in data cleaning, data visualization, and generating actionable business insights
